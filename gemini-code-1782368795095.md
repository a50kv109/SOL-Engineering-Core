# SOL RUNTIME LAYER
**Identifier:** AAM-V1_ARTSYBASHEV_UA_KHARKIV_AIANALYSIS
**Version:** 1.0.0
**Status:** CANONICAL

## Описание
Runtime Layer — это исполнительный механизм реального времени в архитектуре SOL. ACP-Core (Anomaly Control Primitive) является первым официальным модулем данного слоя, представляющим собой **Engineering Reflex Primitive** (виртуальный инженерный измерительный прибор).

Этот слой обеспечивает интеграцию статических знаний (SOL Core, BOOK-NAV, DRAWING-NAV) с динамическими потоками данных (датчики, логи).

## Структура
- `/core/` — замороженный детерминированный математический модуль оценки риска.
- `/epms_templates/` — адаптеры перевода физики в вектор E, V, W, S.
- `/examples/` — демонстрационные скрипты интеграции.
- `/tests/` — модульные тесты для проверки поведения рефлекса.
- `/field_logs/` — директория для сбора статистики и калибровки.

## Интеграция в SOL
Runtime Layer не пересекается с другими слоями, он принимает нормализованные векторы и возвращает дельта-состояния.
