# EPMS SPECIFICATION
**Status:** STABLE

EPMS (Effort, Velocity, Wear, Stability) — стандарт нормализации. Все значения: `0.0 <= x <= 1.0`.

- **Effort (E):** Текущее усилие / Максимально допустимое усилие.
- **Velocity (V):** Текущая скорость изменения / Максимальная скорость.
- **Wear (W):** Накопленная усталость, износ или деградация (0.0 = новый, 1.0 = разрушение).
- **Stability (S):** Мера запаса прочности, отсутствие люфтов или вибраций (1.0 = идеальная стабильность).
