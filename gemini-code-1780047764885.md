# SOL-Core: Базовые примитивы

Фундаментальный словарь протокола AAM-V1.

| Примитив | Назначение |
| :--- | :--- |
| **[NODE]** | Узел взаимодействия. |
| **[PROPAGATE]** | Канал передачи энергии/сил. |
| **[CONSTRAIN]** | Граничные условия. |
| **[STORE]** | Накопление потенциальной энергии. |
| **[DISSIPATE]** | Потери (трение, износ, тепло). |
| **[CONVERT]** | Преобразование вида энергии. |
| **[FEEDBACK]** | Обратная связь. |
