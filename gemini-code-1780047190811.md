## Research Lineage
Этот репозиторий является центральным узлом для разработки протокола **SOL (Structural Operational Language)**.
- **SOL v1.2**: Фундаментальный протокол семантической стабилизации инженерных намерений.
- **SOL-CAD vNext**: Специализированное промышленное расширение для диагностического рантайма и PHM (Prognostics and Health Management).

## Core Architecture
SOL-CAD vNext использует «замороженное» ядро из 7 примитивов, преобразуя статичные CAD-данные в исполняемые графы причинно-следственных связей.
