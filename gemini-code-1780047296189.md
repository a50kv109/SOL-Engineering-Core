# SOL-Core Primitives

Фундамент детерминированной диагностики. Все сложные узлы ([BRG], [GEAR], [MTR]) разлагаются на эти примитивы при анализе отказов.

| Primitive | Definition |
| :--- | :--- |
| **[NODE]** | Точка взаимодействия компонентов. |
| **[PROPAGATE]** | Канал передачи энергии/силы (Rotation/Linear/Flow). |
| **[CONSTRAIN]** | Ограничение степеней свободы (Boundary Condition). |
| **[STORE]** | Накопление потенциальной энергии. |
| **[DISSIPATE]** | Потери (трение, нагрев, износ). |
| **[CONVERT]** | Трансформация вида энергии. |
| **[FEEDBACK]** | Рекурсивный контур контроля. |
