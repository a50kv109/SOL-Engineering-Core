# Architecture: SOL-CAD vNext

SOL-CAD vNext — это промежуточное ПО (Middleware), превращающее статичные CAD-данные в исполняемые графы причинно-следственных связей.

## Уровни абстракции
1. **L1 (SOL-H)**: Инженерная коммуникация.
2. **L2 (SOL-A)**: Протокол взаимодействия между ИИ-агентами.
3. **L3 (SOL-IR)**: Машинно-детерминированное представление (Intermediate Representation).
4. **L4 (SOL-CAD/DW)**: Привязка к топологии САПР (STEP AP242).

## Принципы
- **Топологический детерминизм**: Причинность важнее соглашений об именовании.
- **Минимализм**: Делегирование численной физики внешним решателям.
- **Целостность**: Запрет на «галлюцинации» при неполных данных (использование `[ENTITY:?]`).
